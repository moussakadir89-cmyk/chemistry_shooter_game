🚀 Get Started

Download and play English Learning Shooter in just a few simple steps!

1

Clone Repository

git clone https://github.com/yourusername/english-learning-shooter.git
2

Install Dependencies

pip install -r requirements.txt
3

Run the Game

python english_learning_shooter.py

🎯 Controls

1-4: Select answers | M: Change mode | D: Change difficulty | H: Heal | R: Reset
